# Online Ticket Shop

A web application for browsing, purchasing, and managing tickets for events.
Built with **Django**, this project provides a simple platform where users can view available events, select tickets, and complete purchases online.

## Tech Stack

* Backend: Python + Django
* Database: PostgreSQL
* Frontend: HTML, CSS, JavaScript
* Authentication: Django built-in authentication system
